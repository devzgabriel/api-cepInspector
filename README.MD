## **Como executar na sua máquina e testar**

<br>

### **_Instalando o projeto_**

No terminal digite todos os comandos abaixo:

```Bash
git clone https://github.com/devzgabriel/api-cepInspector.git
```

```Bash
cd api-cepInspector
```

```Bash
yarn install
```

```Bash
yarn dev
```
